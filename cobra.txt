Cobra

Estructura tipica:

cmd/
  launcher/
    main.go
    root.go
    start.go
    stop.go
    status.go
    monitor.go
    config.go


1. Controlar el ciclo de vida del servidor.

launcher start --env prod
launcher stop --graceful
launcher restart --rolling
launcher status
launcher uptime
launcher monitor -> TUI
launcher config set max_players=500 onlyadmin

launcher restart --graceful && launcher monitor


2. Gestión de mapas / mundos
Comandos
launcher world load map_12
launcher world unload map_7
launcher world reload map_5
launcher world list

Scripts
for map in map_1 map_2 map_3; do
  launcher world reload $map
done


Uso:
Actualizar contenido
Hot reload de mapas
Liberar memoria

3. Conexiones y sesiones
Comandos
launcher conn list
launcher conn count
launcher conn kick --account=1234
launcher conn kick --ip=192.168.1.10
launcher conn ban --account=1234 --duration=24h

Scripts
launcher conn kick --inactive --since=30m

Uso:
Moderación
Anti-abuse
Limpieza de sesiones zombies
